1. Prerrequisitos
Tener instalado
1. GIT
2. Python 3.10+
3. Node.js + npm

2. Clonar el repositorio
git clone https://github.com/crgonzalezvi/revo-assistant.git
cd revo-assistant

3. Backend (FastAPI + Gemini + ChromaDB)
3.1. Entrar a la carpeta y crear venv
cd backend
python -m venv .venv

Activar el entorno virtual:

En PowerShell/CMD:
.\.venv\Scripts\activate

En Git Bash:
source .venv/Scripts/activate

3.2. Instalar dependencias
Si existe requirements.txt:

pip install -r requirements.txt


Si no, m√≠nimo:
pip install fastapi uvicorn[standard] python-dotenv chromadb sentence-transformers google-generativeai

3.3. Configurar la API key de Gemini
En backend/ crea un archivo .env con:
GOOGLE_API_KEY=TU_API_KEY_DE_GEMINI_AQUI

3.4. Ejecutar el backend
Con el entorno virtual activado, desde backend/:
python main.py

4. Configurar y ejecutar el frontend
Con el backend corriendo ejecutar:
cd frontend

4.1. Instalar dependencias
npm install

4.2. Levantar el frontend
nom start
